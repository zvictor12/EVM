# Regression Lab

## О задаче

Лабораторная работа №4 по регрессионному анализу. В ноутбуке исследуются два датасета из папки `Datasets` и строятся модели для анализа зависимости между признаками.

## Что сделано

- загружены датасеты `Kuiper` и `cigarettes.dat.txt`;
- подготовлены функции анализа данных;
- выполнен разбор признаков и визуальная диагностика;
- построены регрессионные модели;
- проанализированы результаты и качество аппроксимации.

## Структура

- `topic_4_regression_lab.ipynb` — основной ноутбук с лабораторной;
- `Datasets/` — исходные данные;
- `pyproject.toml` и `uv.lock` — окружение проекта.

## Стек

- Python
- pandas, numpy
- statsmodels
- matplotlib, seaborn
- Jupyter
